```

```

## TC-07 Buff / Debuff 状态效果

### TC-07-A 易伤效果施加与显示

**执行命令**：
```
/e2e 打开 http://localhost:5173，点击"开始游戏"，进入战斗，手牌中找到"猛击"(BASH)卡，选中后点击敌人，截图确认敌人效果区域出现"易伤"状态徽章（红色，显示层数），同时确认敌人 HP 因攻击减少
```

**预期结果**：
- 敌人显示 `易伤 2`（BASH 默认施加 2 层）
- 徽章颜色为红色（debuff 样式）

---

### TC-07-B Debuff 叠加规则：取最大值不叠加

**执行命令**：
```
/e2e 打开 http://localhost:5173，点击"开始游戏"，进入战斗，打出"猛击"使敌人获得易伤(2)，结束回合进入新玩家回合，再次打出"猛击"，截图确认敌人易伤数值为 2（不是 4），证明 debuff 取最大值而非叠加
```

**预期结果**：
- 已有 易伤(2) 的状态下再施加 易伤(2)，结果仍为 `易伤 2`
- 若旧值(2) > 新值，则保持旧值

---

### TC-07-C Buff 效果叠加：力量累加

**执行命令**：
```
/e2e 打开 http://localhost:5173，点击"开始游戏"，进入战斗，出"炽焰"(INFLAME)卡给玩家增加力量，截图记录力量值，再次出"炽焰"（如有），截图确认力量值为两次之和（累加）
```

**预期结果**：
- 力量徽章显示累加后的数值（如 2 + 2 = 4）

---

### TC-07-D Debuff 到期自动移除

**执行命令**：
```
/e2e 打开 http://localhost:5173，点击"开始游戏"，进入战斗，打出"猛击"给敌人施加易伤(2)，结束回合（易伤递减为 1），再结束一回合（易伤递减为 0，应移除），截图确认敌人效果区域易伤徽章已消失
```

**预期结果**：
- 易伤(2) 在两回合后完全移除
- 徽章从敌人效果区消失

---

### TC-07-E Artifact 吸收 Debuff

**前置条件**：玩家需持有 Artifact 效果（或敌人有 ARTIFACT 效果）

**执行命令**：
```
/e2e 打开 http://localhost:5173，点击"开始游戏"（可能需要多次重试找到带 Artifact 的场景），进入战斗，确认目标实体有神器(ARTIFACT)徽章，尝试对其施加一个 debuff，截图确认 Artifact 层数减少 1，debuff 未被实际施加（效果区无新 debuff 徽章）
```

**预期结果**：
- 神器消耗 1 层来免疫 1 次 debuff
- debuff 未生效
